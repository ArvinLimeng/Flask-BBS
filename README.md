# Flask-BBS
#### 项目介绍

一个基于flask的小型bbs论坛系统,支持登录，注册，发布内容，以及评论，内容搜索等功能。
<br>体验地址http://106.12.128.112
#### 所需环境
python版本：3.6.6  
MySQL版本：8.0.12  
所需的包：  flask，os，functools，flask_script，flask_migrate，flask_sqlalchemy
可能还要一些别的包，需要的时候，用pip命令安装即可。
